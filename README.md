# rust-gpwd
随机生成指定长度的字符串

开发中时长使用弱密码，不安全！于是用rust练手，写了这个小工具。

# 使用方法
```
gpwd 18
```

效果：

```

02: 5d706e3f2bc06075ec
03: 76913c3be0137bb608

05: 7a2cc886d1245b49a2
06: 5ba1aba67cf3a86566

08: 9fd2b83b3f15b708fa
09: 0e8fa43bb262184dbe
10: 8ae33de33b50a8f7a4
```
